## App launching
Can be done by pressing right ⌘ + any keys

### Current configuration

#### Left Home row
1. a - ai (chatgpt)
2. s - shell (iterm)
3. d - development (vscode)
4. f - chat app (feishu)
5. g - google chrome

#### Less frequently used
6. m - music
7. l - to do list

### Set up and tweaks

#### Chat gpt ( right ⌘ + A )
